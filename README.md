# modulAR-CR_APP
The https://github.com/MartijnWisse/mirte_workshop was followed for setting up the robot. The respository includes Mirte Master preparations, launch files, arm-gripper movement, navigation, marker palcement, and more. 

# Gripper node was added to Martijn Wisse's example and navigation node was edited from example

# Auto Docking Node was made during project and is available on: https://github.com/deniseheidema/Mirte-Master-Auto-Docking-ArUco

This repository displays the C#- and Python codes that are used in the app and Mirte robot for the creative robotics course 2025-2026

# How To Install?

1. Flash Arduino UNO with _Open_and_close.ino.
2. Place gripper_arduino in ~/mirte_ws/src
3. Place mirte_location_markers folder in ~/mirte_ws/src and replace duplicates
